Project - Movie Trailer Website

What is it?
This project stores a list of my favorite movies and generates a static web
page allowing visitors to browse the movies and watch the trailers.

How it runs?
To start this project, please follow these steps:
    1. Clone this repository.
    2. Open entertainment_center.py in Python idle and click on run module.
    3. If you want to run it from cmd then cd to MovieTrailerWebsite folder and
    run python entertainment_center.py
Your default browser will open the static web page.
